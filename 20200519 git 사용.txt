git commit -m "first time" (깃허브 올릴때 사용할 메시지 주석)
(git hub 에 올릴 환경 사용자 이름)

git remote add origin https://github.com/kty4606/KB.git
(어디서버에 올릴지 지정)
git push origin master
(서버에 올려라)
